1. git init
2. made changes
3. git branch -M main... here renaming the branch 
4. created a git repo in github
    https://github.com/AdityaThakare72/Git_tutorial.git
    we will be commiting here
    git remote add origin https://github.com/AdityaThakare72/Git_tutorial.git

5. u can push the changes to the remote (the one in the github) repo
    git push origin main

    here origin is the url of ur remote repo

6. u can restore the file to the previous stage
    git restore <filename>
